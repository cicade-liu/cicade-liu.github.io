Python与Pytorch

基础学习

详细的入门教程可以查看官网

https://docs.pytorch.org/tutorials/beginner/basics/intro.html

一：基础认识

Python 是通用编程语言，只是提供语法、基础数据类型、文件、循环、函数等基础能力；能写爬虫、后端、数据分析、脚本、AI 等所有程序；

PyTorch 是基于 Python 开发的深度学习框架，底层用 C++/CUDA 实现高性能运算，上层封装 Python API，方便调用；

二：依赖关系

Python解释器 → 导入 torch（PyTorch 库）→ 写神经网络代码

PyTorch 是 Python 的一个扩展工具包，类似：

  • numpy（数值计算库）
  • pandas（数据分析库）
  • requests（爬虫库）
三：各自负责模块

Python 负责：

  • 程序逻辑、循环、判断、函数、类定义；
  • 数据读取、预处理、打印日志、保存文件；
  • 调用 PyTorch 提供的接口。

PyTorch 负责 AI 核心能力：

  1. 张量 Tensor（GPU 加速矩阵运算，替代 numpy）

    张量是 PyTorch 里统一的数据存储格式，可以理解为「支持 GPU 的多维数组」：

    • 0 维：标量（单个数字 5）
    • 1 维：向量 [1,2,3]
    • 2 维：矩阵（图片灰度图、表格数据）
    • 3/4 维：批量图片（[batch,通道,高,宽]）、视频数据

    numpy 只能跑 CPU，速度慢；Tensor 可以一键转移到显卡 GPU，大规模矩阵运算速度提升几十上百倍。

    神经网络所有输入、权重、输出全部都是张量，是整个深度学习计算的基础载体。

  2. 自动微分 autograd（不用手动求导，训练神经网络核心）

    训练网络需要用梯度下降更新参数，数学上要对损失函数求导（算梯度）。

    传统手写程序：需要自己手动推导、编写求导公式，复杂网络几乎不可能实现。

    autograd 自动记录张量所有运算流程，一行代码 loss.backward() 就能自动算出所有参数的导数。

    只要张量设置参数requires_grad=True，PyTorch 就会跟踪它的计算历史，保存计算图，用于反向求导。

  3. 神经网络层：卷积、全连接、激活函数等 torch.nn

    常用组件分类

    1. 网络层（核心结构）

      • 全连接层 nn.Linear：普通线性拟合、分类网络基础
      • 卷积层 nn.Conv2d：图像识别、CNN 专用，提取图片特征
      • 循环层 nn.LSTM/GRU：文本、时序数据处理

    2. 激活函数（给网络增加非线性，否则只能拟合直线)

    3. 容器：nn.Sequential 快速堆叠多层网络

  4. 优化器、损失函数、数据集加载工具

  5. CUDA GPU 加速训练模型

    (1)什么是CUDA

    NVIDIA 显卡的并行计算平台，GPU 拥有上千个小计算核心，擅长大规模张量矩阵运算（神经网络本质就是海量矩阵乘法）。

    CPU 核心少，处理大批量图像 / 模型计算很慢；GPU 并行计算速度提升几十～上百倍。
